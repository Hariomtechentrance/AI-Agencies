Heart Attack Risk Prediction

AI & Machine Learning Project Case Study
#Healthcare #ML #RiskPrediction

Project Overview

This project leverages advanced machine learning algorithms to predict the risk of heart attacks in patients, enabling early intervention and improved healthcare outcomes. Using real-world patient data, the system helps clinicians identify high-risk individuals and reduce misdiagnosis rates.

Problem

Cardiovascular disease is a leading cause of death worldwide. Traditional diagnostic methods can miss subtle warning signs, leading to delayed or incorrect treatment. There is a need for accurate, data-driven tools to assist in early detection and risk assessment.

Solution

The model uses a dataset of 70,000 real patient records. Multiple machine learning algorithms were evaluated, including Decision Trees, Random Forest, XGBoost, and Multilayer Perceptron. Data preprocessing involved outlier removal, feature selection, and clustering to enhance accuracy. The best results were achieved with a Multilayer Perceptron model, which was fine-tuned using cross-validation.

Results

  • Accuracy: 87.28% (Multilayer Perceptron, cross-validated)
  • AUC (Area Under Curve): 0.95
  • Impact: Improved early detection, reduced fatality rates, and better patient outcomes

Technologies Used

  • Python (scikit-learn, XGBoost)
  • Machine Learning: Decision Tree, Random Forest, XGBoost, Multilayer Perceptron
  • Data Preprocessing & Feature Engineering
